# 🎵 Relaxing Audio Player
 
A cross-platform mobile audio player built with **Ionic + Angular** and **Capacitor**. Plays relaxing ambient sounds on Android and iOS.
 
---
 
## Prerequisites
 
Before getting started, make sure you have the following installed:
 
- [Node.js](https://nodejs.org/) (v18 or later recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- [Android Studio](https://developer.android.com/studio) (for Android builds)
- [Java JDK 17+](https://adoptium.net/)
- Android SDK with API level 22+ (installable via Android Studio)
- A physical Android device with **USB Debugging enabled**, or an Android emulator
---
 
## Getting Started
 
### 1. Clone the repository
 
```bash
git clone https://github.com/your-username/Relaxing-Audio-Player.git
cd Relaxing-Audio-Player
```
 
### 2. Install dependencies
 
```bash
npm install
```
 
---
 
## Building & Running on Android
 
### 3. Build the web assets
 
Capacitor requires the web app to be compiled before syncing to the native platform.
 
```bash
npm run build
```
 
This outputs the compiled app to the `www/` directory.
 
### 4. Add the Android platform
 
> **Skip this step** if the `android/` folder already exists in the project.
 
```bash
npx cap add android
```
 
### 5. Sync the web build to the native project
 
Run this after every `npm run build` to copy the latest web assets and update native plugins:
 
```bash
npx cap sync android
```
 
### 6. Run on a connected device
 
Connect your Android device via USB (with USB Debugging enabled) or start an emulator, then run:
 
```bash
npx cap run android
```
 
Capacitor will detect connected devices/emulators and prompt you to choose one. The app will be built and deployed automatically.
 
---
 
## Development Workflow
 
When actively developing, use this sequence to see changes on device:
 
```bash
npm run build && npx cap sync android && npx cap run android
```
 
Or for faster iteration during UI development, use Ionic's live reload:
 
```bash
ionic cap run android --livereload --external
```

## Troubleshooting
 
**`Could not find the web assets directory: .\www`**
Run `npm run build` before `npx cap sync`.
 
**`ERESOLVE unable to resolve dependency tree`**
Make sure all `@capacitor/*` packages are on the same major version. Reinstall cleanly:
```bash
rm -rf node_modules package-lock.json
npm install
```
 
**`EPERM: operation not permitted` on Windows**
Close any editors or tools that may be locking `node_modules`, or run the terminal as Administrator.
 
**USB device not detected**
Ensure USB Debugging is enabled on your device (*Settings → Developer Options → USB Debugging*) and that you've accepted the RSA key prompt on the device.
